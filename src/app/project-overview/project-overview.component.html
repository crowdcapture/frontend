Detecting sigarette butts

trash sigarettes

made by pcmill

Trash is everywhere but especially sigarette butts. Many scientist believe sigarette butts to be the most prevalent sort of trash. With this project I want to built up a dataset that makes it possible to detect sigarette butts. The beautifull thing is almost everyone can contribute to this project by simply going outside and taking some images.
